# Workspace Configuration

## Structure
- `agent-scheduler/` - Node.js scheduling app (Express + node-cron)
- `alarm-app/` - Single-page alarm web app
- `sellpath-lp/` - SellPath landing page (HTML + images)
- `lineworks-monitoring/` - LINE WORKS monitoring script (Python)
- `looker-to-sheets/` - Looker Studio to Google Sheets automation (Python + Selenium + Docker)
- `My-Vault/` - Obsidian notes vault
- `docs/` - Reference documents and articles

## Conventions
- Language: Japanese for user-facing content, English for code and config
- Each project is self-contained in its directory
- Sensitive files (keys, credentials, auth state) are gitignored
