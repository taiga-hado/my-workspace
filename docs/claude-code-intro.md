# Claude Code ってなんなん？めっちゃわかりやすく教えたるわ

## ざっくり言うとな

Claude Code は、**ターミナル（黒い画面）で動く AI アシスタント**やねん。
Anthropic っていう会社が作った Claude の頭脳を、自分のパソコンの開発環境にそのまま連れてこれるんよ。

チャットで「こんなコード書いてや」「このバグなんとかしてくれへん？」って言うだけで、AI がファイル読んで、編集して、コマンドまで叩いてくれるねん。めっちゃ便利やろ？

---

## 今までの AI チャットとなにが違うん？

| 特徴 | Web版 Claude / ChatGPT | Claude Code |
|------|----------------------|-------------|
| 動く場所 | ブラウザ | ターミナル（CLI） |
| ファイル操作 | コピペでがんばる | 直接読み書きできる |
| コマンド実行 | でけへん | `git`, `npm` とか実行できる |
| プロジェクト理解 | ちょっとだけ | フォルダ丸ごと把握できる |

今までの AI チャットやと「コード貼り付けて → 回答コピーして → 自分で反映して…」ってめっちゃ手間やったやん？ Claude Code はその手間を全部すっ飛ばして、**AI が直接コードベースで作業してくれる**のが一番デカいポイントやねん。

---

## なにができるん？

### 1. コードを書いたり直したり

```
> ログイン機能追加してや
```

こう言うだけで、必要なファイル作ったり編集したりしてくれるねん。楽ちんやわ～。

### 2. バグ調査と修正

```
> テスト落ちてるから原因調べて直してくれへん？
```

エラーメッセージ読んで、コード追いかけて、原因突き止めて直してくれるで。探偵みたいなもんやな。

### 3. リファクタリング

```
> この関数ごちゃごちゃしてるから読みやすくしてや
```

今のコードちゃんと理解してから、きれいに整理してくれんねん。

### 4. Git 操作

```
> 変更内容コミットしといて
```

差分確認して、ええ感じのコミットメッセージ考えて実行してくれるで。

### 5. プロジェクト全体の把握

```
> このプロジェクトどんな構成なん？教えてや
```

フォルダ構造とかファイル読み取って、全体像を教えてくれるわ。新しいプロジェクト入った時とかめっちゃ助かるで。

---

## どうやって始めるん？

### その1：インストールするで

```bash
npm install -g @anthropic-ai/claude-code
```

### その2：起動するで

```bash
cd 自分のプロジェクト
claude
```

これだけで会話始まるねん。ほんまにこれだけやで。

### その3：話しかけるだけ

あとは日本語でなんでも言うてみ。

```
> README.md 日本語で書いてや
> src/utils.ts にテスト追加してくれへん？
> このエラーなんでなん？調べてや
```

## 安全なん？大丈夫なん？

ここ気になるよな。安心してや、ちゃんとしてるで。

- **ファイル編集する前に聞いてくれる** — 勝手にコード壊したりせえへん
- **ヤバいコマンドは許可がいる** — `rm` とか `git push` は「これやってええか？」って確認入るで
- **プロジェクトの外には手出さへん** — 作業は自分のプロジェクト内だけや

要するに、AI がやろうとしてることを**自分で確認してからGO**できるから、安心して使えるねん。
